## 🚄 安装

```html
git clone https://github.com/zhengyuanbing/react-ssr

cd react-ssr

npm i
```

## 📦 打包

```html

npm run build

```

## ✈️ 开发

```html

npm run dev

http://localhost:8987 (服务端渲染)

http://localhost:8988 (客户端渲染)

项目通过 nodemon 启动，如果开发过程中 crash 了，按 rs 加回车键就可以重启

```
